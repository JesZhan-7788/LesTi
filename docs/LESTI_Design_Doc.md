## 二、维度系统（4个维度）

| 维度 | 说明 |
|---|---|
| 燃烧 ↔ 克制 | 对爱的强度——全身投入不留退路，还是深爱也保持清醒距离 |
| 边界 ↔ 融合 | 在关系里的自我状态——始终知道自己是谁，还是愿意为爱模糊自己的轮廓 |
| 给予 ↔ 承接 | 爱的方式——主动燃烧、倾尽所有地给，还是静静等待、让爱来找你 |
| 言说 ↔ 沉默 | 爱的语言——用语言、声音、存在感表达，还是用眼神、身体、等待、行动表达 |

## 七、技术实现细节
（略去列表，主要提取计分和距离算法逻辑）
- 计分：同意题(+2, +1, -1, -2)，场景题根据选项计算
- 匹配算法：欧式距离，找出最小距离的人物
- 关键题处理：主维度判定平局特定坐标(Tina/怡敏/Wil)时，比较关键题得分，高者当选。平局优先怡敏。
